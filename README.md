A menu-driven **Address Book Management System** developed in **C** to efficiently store, retrieve, modify, and manage contact information. This project demonstrates the practical implementation of core C programming concepts such as **structures, file handling, modular programming, input validation, and data persistence**.

---

# 📖 Project Overview

Managing contact information manually can become inefficient as the number of contacts grows. This project provides a lightweight, console-based solution for organizing contacts while strengthening the fundamentals of C programming through a real-world application.

---

# ✨ Features

- ➕ Add a new contact
- 📋 View all saved contacts
- 🔍 Search contacts by name, phone number, or email
- ✏️ Edit existing contact details
- ❌ Delete contacts
- 💾 Save contacts to a file for permanent storage
- 📂 Automatically load saved contacts at startup
- ✅ Validate phone numbers and email addresses
- 📌 User-friendly menu-driven interface

---

# ⚙️ Workflow

```text
Start Program
      │
      ▼
Load Existing Contacts
      │
      ▼
Display Main Menu
      │
      ├── Add Contact
      ├── View Contacts
      ├── Search Contact
      ├── Edit Contact
      ├── Delete Contact
      ├── Save Contacts
      └── Exit
```

# 📥 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AddressBook.git
```

Move into the project directory

```bash
cd AddressBook
```

Compile using GCC

```bash
gcc *.c -o addressbook
```

Or compile using Makefile

```bash
make
```

---

# ▶️ Run the Program

### Linux / macOS

```bash
./addressbook
```

### Windows

```bash
addressbook.exe
```
